## 主要更新内容

### 修复 app 内更新安装失败
- 移除 PackageInstaller Session API（静默失败导致安装界面不弹出）
- 恢复 Intent 安装方式，修复下载完成后无安装提示的问题
- 优化安装流程：先调起系统安装器，成功后再关闭下载对话框
- 安装失败时在对话框中显示具体错误信息

### 优化 tab 切换性能
- 关注列表/收藏夹/稍后再看使用 IndexedStack，切换时保持各 tab 状态和滚动位置
- 设置页面各子 tab 使用 IndexedStack，避免切换时重复加载设备信息、缓存大小等数据
- 每个子 tab 使用独立 ScrollController，互不干扰

### GitHub Actions 改进
- 支持手动触发构建（workflow_dispatch），可指定任意 tag 重新构建
- 构建产物（APK）同时上传到 Artifacts，方便直接下载
