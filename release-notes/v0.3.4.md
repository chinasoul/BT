## 主要更新内容

### 应用内更新优化
- 新增 Gitee 备用下载源，中国大陆用户自动从 Gitee 下载，无需任何配置
- 更新检查自动选择最佳源：GitHub → Gitee，全程透明
- GitHub 下载失败自动回退到代理镜像加速
- APK 架构匹配改用正则模糊匹配，兼容各种文件命名（v7a/v8a/arm64 等）
- 下载弹窗实时显示当前下载链接，方便排查问题

### 自动检查更新
- 新增自动定期检查更新功能（关闭 / 每天 / 每3天 / 每7天）
- 有新版本时启动 app 自动弹窗提醒，无更新不打扰
- 设置页显示上次检查时间

### CI/CD
- Release 构建自动同步发布到 Gitee Release

### 其他
- UI 组件优化、设置界面调整
